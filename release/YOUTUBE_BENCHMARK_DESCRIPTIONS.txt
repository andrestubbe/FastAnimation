TITLE:
1 Million Animations in Java?! 🚀 FastAnimation Native Benchmark

DESCRIPTION:
FastAnimation is an ultra-fast timeline engine built for zero-latency UI transitions and complex motion graphics in Java. By bypassing standard Java Thread.sleep limits and integrating native Windows Multimedia Timers (via FastDWM), we unlock massive performance gains and buttery smooth 1000+ FPS updates for extreme GUI stress tests.

KEY HIGHLIGHTS:
- High-Precision Timing: Sub-millisecond updates using a dedicated native engine thread (NATIVE_MM).
- 1 Million Tweens: Effortlessly handles 1,000,000 parallel animations.
- Zero GC Pressure: Memory-pool architecture eliminates micro-stutters during heavy GC stress.

RESULTS (1M Tweens / GC Stress):
- Standard Java: ~180 FPS (Max Jitter: ~4359 μs)
- FastAnimation Native: ~560 FPS (Max Jitter: ~3258 μs)
- Speedup: 3.1x faster Tick Rate & 25% smoother under garbage collection stress!
