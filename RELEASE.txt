## FastAnimation v1.0.0 - Initial Release

**Lightweight Java timeline engine for orchestrating multiple tweens and keyframes.** Built on [FastTween](https://github.com/andrestubbe/FastTween) with integrated ticker. Sequences, parallel execution, loops, events — zero overhead.

### Features
- ⚡ **Sequences** — Chain tweens one after another
- 🔄 **Parallel** — Run tweens simultaneously  
- ♻️ **Loops** — Repeat with count or infinite (-1)
- 🎬 **Keyframes** — Define progress points (0% → 30% → 100%)
- 🔔 **Events** — onStart, onUpdate, onComplete hooks
- ⏱️ **Built-in Ticker** — Automatic time management via daemon thread
- 🔗 **Built on FastTween** — Zero overhead, pure Java

### Installation

### Quick Start
```java
// Sequence of tweens
FastAnimation anim = FastAnimation.sequence(
    FastTween.to(0f, 100f, 200).ease(Ease.QUAD_OUT),
    FastTween.to(100f, 20f, 400).ease(Ease.CUBIC_IN_OUT)
).loop(3).start();

// Parallel execution
FastAnimation parallel = FastAnimation.parallel(
    FastTween.to(x0, x1, 300).onUpdate(x -> setX(x)),
    FastTween.to(y0, y1, 300).onUpdate(y -> setY(y))
).start();
```
